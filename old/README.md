## Getting Started

**Run the development server:**

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

**Build for production:**

```bash
yarn build
```

## Deployment

Steps:
- Push to `main` branch

Vercel will deploy automatically.
